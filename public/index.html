Accueil

Bienvenue 👋 Cette application vous aide à explorer vos ressentis pas à pas.

Tuto ultra simple : vous activez des mots, vous déplacez les nœuds, puis vous créez des liens entre eux par contact.

  • Lier : faites glisser un nœud jusqu’à toucher un autre nœud. Un lien se crée automatiquement.
  • Délier : touchez le trait entre deux nœuds pour retirer ce lien.
  • Déselectionner un mot : touchez à nouveau son bouton, le nœud et ses liens disparaissent.

Les pages du menu :

  • 🏠 Accueil : mode d’emploi rapide de l’appli.
  • 🎧 Écouter : parcours QR, étape par étape.
  • 🕸️ Tisser : espace principal pour poser et relier vos ressentis.
  • 📄 Synthétiser : résumé vivant de votre tissage.
